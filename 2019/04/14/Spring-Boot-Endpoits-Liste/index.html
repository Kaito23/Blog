Spring Boot - Endpoint Liste

14-04-2019

Ich habe mich mal wieder ein bisschen mit Spring (Boot) beschäftigt. Aktuell probiere ich eine kleine Datenbank-Anwendung mit (wie Üblich) einem VUE.js Frontend aus. Als unterstützung während der Programmierung hätte ich gerne eine Übersicht über die Endpoints gehabt, die ich in meiner Anwendung erstellt habe.
Bei der Suche nach einer Lösung stolperte ich zuerst über Spring Actuator. Dieser Lieferte mir auch mit der Methode “Mappings” eine Schnittstelle, die mir die Endpoints meiner Anwendung ausgab … eine große Menge Endpoints. Diese Liste ließ sich auch leider durch keine API-Methoden einschränken. Ich habe mir dann eine eigene Schnittstelle erstellt. Diese Liefert mir eine Liste aller Endpoints mit folgenden Informationen:

  • Path: Pfad in der URL
  • Type: Typ (POST | GET)
  • Parameters: Anzahl der Parameter
  • Class: die Klasse
  • Method name: Name der Methode ind er Klasse

Die Sourcen habe ich heute mal bei Gist eingestellt:
https://gist.github.com/Kaito23/4d09a6f68c57db042ba5226e97d7b716

Weitere Informationen zu Actuator und den Verwendeten Methoden knnen hier gefunden werden:

  • https://docs.spring.io/spring-boot/docs/current/reference/html/production-ready-endpoints.html#production-ready-endpoints-custom

  • https://docs.spring.io/spring-boot/docs/current/actuator-api/html/#mappings

Aktuell prüfe ich noch die Erweiterbarkeit um z. B. die Einbindung des JavaDocs um gleichzeitig eine Dokumentation zu haben.
